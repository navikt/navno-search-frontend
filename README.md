React/[Next.js](https://nextjs.org/) frontend for søk på nav.no 

## Lokal kjøring

Kjører lokalt på [http://localhost:3001](http://localhost:3001).

#### Development mode:
```
npm run dev
```

#### Production mode:
Kopier innhold fra .env.development til .env.local og kjør:

```
npm run build && npm start
```
